## 🚀 Project Overview
This project is a functional authentication system that handles user registration (Sign-Up) and access control (Sign-In). It features a robust validation system and an automated secure password generator. Developed as part of my **Full Stack & AI** path at **Conquer Blocks Academy**.

## 🛠️ Features
- **Modular Architecture:** Logic separated into specialized modules (`validator_usersdata`, `password_generation`).
- **Secure Password Generation:** Uses Python's `secrets` module for cryptographically strong passwords.
- **Complex Validation:** Ensures usernames are unique and passwords meet high-security standards.
- **Interactive CLI:** A user-friendly menu-driven interface.

## 📂 File Structure
- `main.py`: Manages the main loop and user interaction.
- `validator_usersdata.py`: Logic for credential validation.
- `password_generation.py`: Utility for generating randomized secure passwords.
